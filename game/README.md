# Game Project

This is the Godot project root for the reusable 3D game development foundation.

No gameplay, theme, final levels, final player controller, or game-specific assets should be added during the foundation phase.

Current reusable system areas:

- `systems/camera/`: MMO-style third-person camera foundation.
- `systems/character/`: neutral character movement foundation.
- `systems/controls/`: MMO-style controls feel coordination.
